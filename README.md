# Simple Web Portfolio

A simple resume and portfolio page to teach introductory HTML and CSS to students with no previous coding experience.

## HTML Topics Covered

* paragraphs
* headings
* links
* lists
* divs
* classes
* IDs
* Unicode

## CSS Topics Covered

* font-styles
* alignment
* colors (plain-text and hex)
* borders
* margins
* padding
* floats
* width and height
* list styles
* link styles
